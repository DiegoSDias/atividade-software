# Atividade: Repositório de Códigos e Versionamento
## Curso: Sistemas de Informação

Este repositório foi criado para a atividade prática de Engenharia de Software. O objetivo é demonstrar o domínio sobre ferramentas de versionamento (Git) e gestão de repositórios remotos.

### Conteúdo do Repositório:
- **Definições de Git:** Documentação dos termos técnicos essenciais.
- **index.php:** Uma página simples em PHP para demonstrar a persistência e visualização do código no ar.

### Importância do Versionamento:
A ausência de um padrão de versionamento é um dos principais problemas identificados em sistemas legados, como o SIACAD, gerando falta de controle sobre mudanças e erros frequentes em produção[cite: 1]. Através deste repositório, aplica-se o controle de branches e commits para garantir a manutenibilidade do software[cite: 2].
